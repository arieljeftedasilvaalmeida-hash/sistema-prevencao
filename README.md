# 🩺 Sistema Inteligente de Prevenção em Saúde

Aplicativo desenvolvido para auxiliar no cuidado diário com a saúde, funcionando como um assistente pessoal. O sistema permite gerenciar medicamentos, acompanhar vacinas, receber alertas automáticos e visualizar o histórico de cuidados.

Além disso, conta com análise de risco baseada no comportamento do usuário e geração de relatórios em PDF.

---

## 🚀 Tecnologias utilizadas

- Angular
- TypeScript
- HTML
- CSS

---

## 📦 Como rodar o projeto

### 🔧 Pré-requisitos
- Node.js instalado
- Angular CLI instalado

Caso não tenha o Angular CLI:
```bash
npm install -g @angular/cli
```

---

### ▶️ Passo a passo

1. Clone o repositório:
```bash
git clone https://github.com/arieljeftedasilvaalmeida-hash/sistema-prevencao.git
```

2. Acesse a pasta do front-end:
```bash
cd sistema-prevencao/front-end
```

3. Instale as dependências:
```bash
npm install
```

4. Execute o projeto:
```bash
ng serve
```

5. Acesse no navegador:
http://localhost:4200

---

## 📌 Funcionalidades

- Controle de medicamentos
- Acompanhamento de vacinas
- Sistema de alertas inteligentes
- Histórico de atividades
- Análise de risco
- Geração de relatórios em PDF
